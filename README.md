### Probar Localmente

```bash
# Ejecutar pruebas locales
test-local.bat

# Simular pipeline completo
simulacion-pipeline.bat

# Simular con deploy
simulacion-pipeline.bat --deploy
```

### Comandos Terraform

```bash
# Inicializar
terraform init

# Validar
terraform validate

# Planificar
terraform plan

# Aplicar (solo si tienes AWS configurado)
terraform apply
```
